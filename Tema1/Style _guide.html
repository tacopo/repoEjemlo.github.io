What is a Style Guide

A style guide is a document that allows any developer to know the basic information abotu the style of a web site.

Why shoud we use it

We should use it because we need to be compenetrated with our workmates.

Basic rules in style guide

  • Must be intuitive
  • Must be easy
  • Must be informative
  • Must be compatible with CSS

Elements in style guide

  • Design
  • Usability and readability
  • Color palette
  • Fonts
  • Navigation elements sytle
  • Images
  • Others
